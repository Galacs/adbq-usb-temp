# ADBQ USB temperature sensor

ESP32-S3 based temperature USB-C compact sensor board made to test small step-down power supply design, i2c ambient temperature sensors, led segment displays and custom class USB peripherals.

## IC List

- ESP32-S3 (Main MCU)
- 2x TMP117 (I2C temperature sensor)
- TPS6282 (Low Iq buck converter)
- 2450AT18A100E (2.45GHz Ceramic chip antenna)
- XL-SB2301SRWC (SMD 3 digit segment display)
